# ASP.NETCore-App-V1
An ASP.NET Core app based on the ASP.NET Core Fundamentals course at Pluralsight working with:

- Razor pages
- Page models
- Dependency injection
- Forms
- Tag helpers
- Page routing
- GET-POST methods
- TempData
- DataAnnotations and ModelState
- AppSettings configuration
- EF Core (Migrations) 
- Dotnet CLI
- Partial views
- View Components
- Static files
- Middleware
- IIS to publish
